# 科目A免除試験 対策問題集

サーティファイ「科目A免除修了試験」（基本情報技術者試験 科目A 相当）の対策用 Web アプリ（フレームワーク不要の静的サイト）。

## 機能
- 📚 カテゴリ別練習（テクノロジ系／マネジメント系／ストラテジ系／すべて・出題数10/20/全）
- 📝 模擬試験（全分野からランダム50問・合格ライン60%判定）
- 解答後にすぐ正誤判定＋解説、選択肢は毎回シャッフル
- 🎯 結果画面に「得意・苦手レーダーチャート」（10分野）と分野別正答率（苦手な順）
- 📤 結果の共有（スマホは共有シート、PCはクリップボードコピー）
- 間違えた問題だけの復習

## 使い方
`index.html` をブラウザで開くだけ。ビルド不要。

## 構成
- `index.html` … アプリ本体（バニラ JS・レーダーチャートSVG描画含む）
- `data.js` … 問題データ（`window.QUESTIONS`）
- `_server.ps1` … ローカルプレビュー用の簡易静的サーバ（デプロイ対象外）

## 著作権
問題は本アプリ用に新規作成したオリジナルです。出題範囲は基本情報技術者試験 科目A に準拠しています。学習目的で利用してください。
